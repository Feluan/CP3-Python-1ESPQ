📸 JOVI Academy - Controle de Avaliações
🧠 Sobre o Programa

Este programa simula um sistema da câmera JOVI Academy Mode, responsável por registrar avaliações de qualidade de tradução (de 0 a 100).

O usuário insere várias notas, que são armazenadas em uma lista. Ao final, o sistema processa essas informações e gera um relatório contendo:

📊 Média das avaliações
🔢 Quantidade total de registros
⭐ Desempenho geral das traduções
⚙️ Funcionalidades Utilizadas
⌨️ Entrada de Dados
input()
Permite que o usuário digite as avaliações diretamente no terminal.
📥 Armazenamento
append()
Adiciona cada nova nota na lista avaliacoes, permitindo guardar múltiplos valores.
📊 Processamento de Dados
len()
Retorna a quantidade de avaliações registradas.
sum()
Soma todas as notas da lista, sendo usada para calcular a média.
for
Percorre os valores da lista para realizar análises, como contar quantas notas são maiores ou iguais a 80.
📈 Organização dos Resultados
sort()
Ordena as avaliações em ordem decrescente, facilitando identificar a melhor e a pior nota.
🚀 Resultado Final

Ao executar o programa, o usuário recebe um relatório completo no terminal com todas as informações organizadas, simulando o funcionamento de análise de desempenho da câmera JOVI.
